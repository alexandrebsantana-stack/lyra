# 🌌 LYRA

**Nem toda ajuda faz barulho.**

LYRA é uma plataforma digital desenvolvida com o objetivo de oferecer suporte seguro, rápido e discreto para pessoas em situação de violência doméstica. A aplicação utiliza tecnologia para conectar usuários a redes de apoio, garantindo proteção e acessibilidade em momentos críticos.

---

## 🧠 Sobre o Projeto

A violência doméstica é um problema social grave que muitas vezes impede as vítimas de buscarem ajuda por medo, vigilância constante ou falta de acesso a recursos.

O LYRA surge como uma solução tecnológica que permite solicitar ajuda de forma **silenciosa, rápida e segura**, priorizando a discrição e a proteção do usuário.

---

## 🎯 Objetivo

Desenvolver uma aplicação capaz de:

- Permitir pedidos de ajuda em poucos segundos  
- Garantir discrição no uso  
- Facilitar o contato com redes de apoio  
- Oferecer informações úteis e acessíveis  
- Reduzir o tempo de resposta em situações de risco  

---

## 💡 Funcionalidades

- 🚨 Botão de emergência  
- 📍 Envio de localização em tempo real  
- 💬 Chat seguro e discreto  
- 📚 Informações jurídicas e de apoio  
- 🤝 Integração com contatos de emergência  
- 🏥 Acesso a serviços (delegacias, ONGs, hospitais)

## 🛡️ UX/UI

O design do LYRA foi desenvolvido com foco em:

- Discrição total  
- Navegação simples  
- Ações rápidas  
- Interface intuitiva  
- Segurança do usuário  

### Fluxo principal:

1. Usuário acessa o sistema  
2. Aciona o botão de ajuda  
3. Alerta é enviado automaticamente  
4. Localização é compartilhada
